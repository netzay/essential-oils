# Essential Oils API (Backend)

This app provides a quick way to store and view essential oils and their details and benefits.


## Installation

    Clone repo - run rails s -p 3001 from terminal

## Usage

- View/delete oils.
- Add new oils:
  - name 
  - uses
  - description
  - image
  
## Built with

	Ruby version: 2.6.1p33
